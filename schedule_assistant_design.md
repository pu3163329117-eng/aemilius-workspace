## 📱 用户交互设计

### 输入方式
1. **自然语言输入**
   ```
   你: "明天下午2点和技术团队开会"
   助手: "✅ 已安排：明天14:00-15:00 技术团队会议"
   ```

2. **快速命令**
   ```
   /schedule 明天14:00 会议 技术团队
   /remind 周三前 完成项目计划
   /check 今天日程
   ```

3. **对话式管理**
   ```
   你: "我这周有什么重要安排？"
   ```

### 数据存储设计
```sql
-- 日程表
CREATE TABLE schedules (
    id UUID PRIMARY KEY,
    user_id VARCHAR(100),
    title VARCHAR(255),
    description TEXT,
    start_time TIMESTAMP,
    end_time TIMESTAMP,
    priority INTEGER,
    reminder_settings JSONB,
    status VARCHAR(50)
);

-- 任务表
CREATE TABLE tasks (
    id UUID PRIMARY KEY,
    schedule_id UUID REFERENCES schedules(id),
    description TEXT,
    assignee VARCHAR(100),
    deadline TIMESTAMP,
    status VARCHAR(50)
);
```
