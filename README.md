# Aemilius Workspace

这是文歆（Aemilius）的工作空间，包含 AI 助手工具、项目管理脚本和各种技能。

## 项目概述

这个仓库是 AI 助手文歆（Aemilius）的工作空间，包含以下内容：

### 核心文件
- `IDENTITY.md` - 文歆的身份定义
- `USER.md` - 用户信息（Rain）
- `SOUL.md` - 文歆的核心价值观和行为准则
- `AGENTS.md` - 工作空间指南
- `TOOLS.md` - 本地工具配置

### AI 助手工具
- `ai_assistant_builder.py` - AI 助手构建器
- `schedule_assistant.py` - 日程管理助手
- `smart_reminder_system.py` - 智能提醒系统
- `income_tracker.py` - 收入跟踪器
- `project_dashboard.py` - 项目仪表板

### 项目管理
- `project_management_plan.md` - 项目管理计划
- `project_evaluation.md` - 项目评估
- `work_progress_tracker.md` - 工作进度跟踪
- `github_projects_database.md` - GitHub 项目数据库

### 技能库 (Skills)
包含各种 AI 技能，用于扩展功能：
- 文档处理（docx, pdf, pptx, xlsx）
- 前端设计
- 内部通讯
- MCP 构建器

- 等等...

### 项目文档
- `NS_Smart_Shopping_九宫格战略分析.md` - NS Smart Shopping 项目分析
- `cherry_studio_analysis.md` - Cherry Studio 分析
- `technical_architecture_design.md` - 技术架构设计
- `marketing_plan.md` - 营销计划

## 使用说明

1. **环境要求**：
   - Python 3.8+
   - Node.js (部分技能需要)
   - Git

2. **快速开始**：
   ```bash
   # 克隆仓库
   git clone <repository-url>
   
   # 查看可用工具
   ls *.py *.sh
   
   # 运行 AI 助手
   python ai_assistant_builder.py
   ```

3. **技能使用**：
   技能位于 `skills/` 目录，每个技能都有对应的 `SKILL.md` 文件说明使用方法。
