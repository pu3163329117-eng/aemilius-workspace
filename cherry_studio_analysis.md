## 🏗️ 架构分析

### 技术栈概览
- **桌面框架**: Electron + Vite
- **前端**: React + TypeScript
- **构建工具**: electron-vite
- **包管理**: pnpm (monorepo)
- **数据库**: SQLite + Drizzle ORM
- **测试**: Vitest + Playwright
- **代码质量**: Biome + ESLint + TypeScript

### 前端架构
- **框架**: React 18 + TypeScript
- **状态管理**: 待分析（可能是 Zustand/Redux）
- **UI组件**: 自定义组件 + 可能使用Tailwind
- **构建工具**: Vite + electron-vite
- **国际化**: 支持20+语言

### 后端架构  
- **运行时**: Node.js >= 24.11.1
- **API框架**: Electron IPC通信
- **数据库**: SQLite（本地存储）+ Drizzle ORM
- **文件管理**: WebDAV支持
- **AI集成**: 多模型API集成

### 项目结构
```
cherry-studio/
├── packages/          # Monorepo包
├── src/              # 源代码
│   ├── main/         # Electron主进程
│   ├── renderer/     # 渲染进程
│   └── preload/      # 预加载脚本
├── docs/             # 文档
├── tests/            # 测试
├── scripts/          # 构建脚本
└── resources/        # 资源文件
```
