## 关键计算公式

1. **用户获取成本（CAC）** = 总营销成本 / 新增用户数
2. **月活跃用户率（MAU）** = 月活跃用户数 / 总用户数
3. **付费转化率** = 付费用户数 / 总用户数
4. **客单价（ARPU）** = 总收入 / 付费用户数
5. **用户留存率** = 第N日仍在使用的用户数 / 第1日新增用户数
6. **投资回报率（ROI）** = (总收入 - 总成本) / 总成本
